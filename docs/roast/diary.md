### `2023.03.19 杭州 🌤️` 🤨

前两天 **ChatGPT-4** 发布了，只能说震撼，很多人说 **CV** 和 **NLP** 等国内人工智能领域已经凉凉了，这方面研究生没前途了，但我并不这么认为，作为一个经历过美国这种毫无底线的拿政治干涉商业、科技等发展的事件之后的国家，我们应当深知被掐脖子的憋屈与无奈，我们当有卧薪尝胆之资，越是这种关键的技术，我们越要迎头赶上，越要自己掌握，越要实现领先，诚然 OpenAI 直接给出了相对研发而言物美价廉的 API 调用方案，我们也理解公司的基本目的在于盈利，并且 **大规模多模态模型** 我们差的不仅在模型算法，还在数据集，还在算力上，可以说国内没有几个公司有这个能力去负担。但雷总也说过，“成功的公司赚钱利润，伟大的公司赢得民心！”，伟大的公司应当是具备强烈的社会责任感和家国情怀，而我们的政府就算不在明面上宣扬，也当在暗地里提供更多支持，当今世界，软实力的地位将愈发身高，何况这一变革型的技术，将带动的是一整个 软件、模型算法、云计算（算力）产业的协同生态，这将带来的赋能效益也将是无与伦比的……

今天在刚在打字时，**微软输入法**（当时用它是因为好看，虽然没搜狗好用，然而当初的卡顿等问题都已经不存在，且配合自带的剪切板、emoji 等用起来真的越来越舒服了）弹出提示是否打开 **Bing 输入提示**，联想到最近微软还宣布了将 ChatGPT-4 全方位融入 **Office365**、**New Bing**，就不免得不寒而栗，加之还收购了 **OpenAI**、**Github** 等，不得不说，这波微软说不定真的会引领行业发展，至于能不能引发苹果当年的效应，让我们拭目以待吧！

我们也要加油呀！！！！
